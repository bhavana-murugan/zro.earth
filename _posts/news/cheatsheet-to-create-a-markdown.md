# Emphasis
```
*This text will be italic*
_This text will be crossed out (strikethrough)_
**This text will be bold**
_You **can** also combine them_
***All this text is bold and italic***
```

# Lists
## Unordered
```
- Item 1
- Item 2
  - Item 2.1
    - Item 2.1a
    - Item 2.1b
  - Item 2.2
```

## Ordered
```
1. Item 1
2. Item 2
  2.1. Item 2.1
    2.1.a Item 2.1a
    2.2.b Item 2.1b
  2.2. Item 2.2
```
